Narrative Project – OOP

First object-oriented programming project built in C#.

Overview

This Narrative Project was created for my first Object-Oriented Programming course and represents my very first real programming project built using classes and objects.

The goal of the project was to apply the fundamentals of object-oriented design in a practical way by building a narrative-driven application where the player progresses through a story using structured classes, methods, and interactions.

My Role

  • Designed and implemented all core classes and objects.
  • Created the narrative flow using object interactions.
  • Handled user input and story progression logic.
  • Managed game state and branching outcomes.

Technical Details

  • Language: C#
  • Main Focus: Object-Oriented Programming
  • Concepts Used: Classes, Objects, Encapsulation, Methods

Key Systems

  • Narrative system driven by object interactions.
  • State tracking to manage story progression.
  • User choice handling through structured methods.

What I Learned

  • How to design and use classes effectively.
  • The importance of encapsulation and clean object structure.
  • How to organize a project using multiple related objects.
  • How program logic flows between interacting objects.
