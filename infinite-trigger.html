Infinite Trigger – Shooter Game

Third-person wave-based shooter built in Unreal Engine (Blueprints).

Overview

Infinite Trigger is a third-person shooter where the player fights against waves of enemies in an arena-style environment. The game focuses on responsive controls, satisfying abilities, and clear visual feedback.

My Role

  • Designed and implemented core gameplay systems in Blueprints.
  • Created wave-spawning logic and enemy AI behaviour using navmesh.
  • Handled UI elements such as health, wave indicators, and basic menus.

Technical Details

  • Engine: Unreal Engine
  • Scripting: Blueprints
  • Tools: Unreal Editor

Key Systems

  • Wave manager that spawns enemies based on difficulty scaling.
  • Enemy AI using navmesh for pathfinding and chasing the player.
  • Ability system for dash, time slow, and double jump with cooldowns.
